jQuery mobile dynamic popup()

Plugin description

$.dynamic_popup() is a light-weight jQuery Mobile plugin based on the popup() function available since 1.2 alpha. It enables you to create a dynamic popup without stressing you with details where to put the actual HTML code. This is extremely useful if you have many dynamic pages in your jQuery Mobile web application.

Read my original blog post about the plugin and also the GitHub feature "Add ability to make popups common to multiple pages".
